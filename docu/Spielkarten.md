
## Status der NSCs

Der Status der NSCs kann mit kleinen Spielsteien auf der Karte markiert werden.

| Symbol | Name | Beschreibung | Statuswechsel |
|---|---|---|---|
| ![Abgelenkt](.attachments/Abgelenkt.svg)| Abgelenkt | −2 auf alle Eigenschaftsproben. | Endet am Ende des nächsten Zuges des Betroffenen. |
| ![Angeschlagen](.attachments/Angeschlagen.svg)| Angeschlagen | Es sind nur freie Aktionen (einschließlich Bewegung) möglich. | Wird zu Beginn des eigenen Zuges automatisch aufgehoben oder sofort durch eine Willenskraftprobe als Aktion. |
| ![Benommen](.attachments/Benommen.svg)| Benommen | Abgelenkt und Verwundbar; keine Bewegung und keine Aktionen möglich. | Zu Beginn des eigenen Zuges Willenskraftprobe: Erfolg hebt Benommen auf, eine Steigerung erlaubt außerdem normales Handeln in diesem Zug. |
| ![Festgehalten](.attachments/Festgehalten.svg)| Festgehalten | −2 auf die Bewegungsweite (mindestens 0); Rennen ist nicht möglich. | Als Aktion Athletik- oder Stärkeprobe: Erfolg hebt Festgehalten auf. |
| ![Gebunden](.attachments/Gebunden.svg)| Gebunden | Keine Bewegung, −2 Parade und keine körperlichen Aktionen außer dem Befreiungsversuch. | Als Aktion Athletik- oder Stärkeprobe mit −2: Erfolg reduziert den Zustand auf Festgehalten, eine Steigerung hebt ihn ganz auf. |
| ![Verwundbar](.attachments/Verwundbar.svg)| Verwundbar | Angreifer erhalten +2 auf Proben, die den Betroffenen unmittelbar beeinflussen. | Endet am Ende des nächsten Zuges des Betroffenen. |

## Weitere Symbole

Weitere Symbole und deren Bedeutung.

| Symbol | Name | Beschreibung |
|---|---|---|
| ![Parade](.attachments/Parade.svg)| Parade | Der Paradewert muss bei einem Nahkampfangriff überwunden werden. |
| ![Geschwindigkeit](.attachments/Geschwindigkeit.svg)| Geschwindigkeit | Normale Bewegungsweite auf dem Spielfeld. |
| ![Robustheit](.attachments/Robustheit.svg)| Robustheit / Rüstung | Bei Schaden wird die Robustheit zusammen mit dem Rüstungswert von den Schadenspunkten abgezogen. Panzerbrechende Waffen durchdringen den Rüstungswert, haben aber keinen Einfluss auf die Robustheit.  |
| ![Team](.attachments/Team.svg)| Gruppe Söldner | Der NSC benötigt eine Gruppe von Söldnern. Weitere NSCs müssen von dem Söldner Stapel gezogen werden. |
| ![Team](.attachments/Team_Kultisten.svg)| Gruppe Kultisten | Der NSC benötigt eine Gruppe von Kultisten. Weitere NSCs müssen von dem Kultisten Stapel gezogen werden. |
| ![Team](.attachments/Team_Schläger.svg)| Gruppe Schläger | Der NSC benötigt eine Gruppe von Schlägern. Weitere NSCs müssen von dem Schläger Stapel gezogen werden. |
| ![Team](.attachments/Team_Bestien.svg)| Gruppe Bestien | Der NSC benötigt / beschwört eine oder mehrere Bestien. Diese müssen von dem Bestien-Stapel gezogen werden.  |
