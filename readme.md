Dieses Projekt erstellt mit Hilfe einer SVG-Vorlage und einer CSV-tabelle eine mehrseitige Pdf-Datei

Textersetzungen:

## Allgemein

| Feld | data-field | Typ |
|---|---|---|
| Name | name | text |
| Level | level | Zahl |
| Level | level_color | Farbe |
| Level | level_back_color | Farbe |
| Parade | parry | zahl |
| Robustheit | thoughness | zahl + () |

## Eigenschaften 

| Feld | data-field | Typ |
|---|---|---|
| Geschicklichkeit | agility | würfelgruppe |
| Verstand | smarts | würfelgruppe (W4-W12) |
| Stärke | strength | würfelgruppe |
| Konstitution | vigor | würfelgruppe |
| Willenskraft | spirit | würfelgruppe |
